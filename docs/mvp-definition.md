# MVP Definition

## Customer

The first customer is a small to mid-sized custom cabinet shop that needs better visibility into active jobs and department handoffs.

## Use Cases

- Create and manage cabinet projects.
- Track each project through the shop workflow.
- Assign tasks to departments or users.
- Store drawings, approvals, photos, and job files.
- See what is blocked, overdue, or ready for the next step.
- Log notes, updates, and job history.

## Features

- User authentication
- Role-based access for owners/admins, managers, department leads, and employee contributors
- Project list and project detail pages
- Client records with active files and archived project history
- Project status and department stage tracking
- Task model with department ownership
- Shop Floor, Sales, Design, and Engineering work views
- Project PDF upload, metadata editing, deletion, viewing, and download with manager-only write access
- Dashboard notification bell and side panel for task assignments, project changes, blocked work, file uploads, and other important updates
- Time logging for tasks or departments

## Roles

- Owner/Admin
- Manager
- Sales
- Designer
- Purchaser
- Shop Lead
- Department User
- Installer

## Success Metrics

- Shops can see every active job and current department status.
- Managers can identify blocked or overdue work quickly.
- Teams reduce status-check messages and manual follow-ups.
- Project files and approvals are easier to find.
- Department handoffs become clearer and more consistent.

## Non-Goals

- Full accounting system
- Full CRM for all lead-generation activity
- Full estimating engine
- Payroll system
- Generic construction project management platform
- Deep ERP replacement in the first release

## Future Specs To Track

- Add worker clock-in and clock-out flows for cabinet-shop labor time.
- Track project revenue and profitability so managers can see how many dollars each project made.
- Add role-specific views and access controls, including manager access to the full dashboard and restricted worker access without the dashboard view.
